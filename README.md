# Monde Perdu — Téléchargements

Ce dépôt sert uniquement à distribuer les versions installables de **Monde Perdu**, un jeu d'aventure 3D multijoueur sur île procédurale.

Aucun code source n'est hébergé ici — va dans l'onglet [Releases](../../releases) pour télécharger la dernière version (`.exe`, Windows).

L'application se met à jour automatiquement.
